* Load and explore real-world data
* Clean and prepare metadata
* Create meaningful visualizations
* Build a simple interactive app
* Document findings and reflections

---

## 🚀 Setup & Installation

1. Clone this repo:

```bash
git clone https://github.com/YourUsername/Frameworks_Assignment.git
cd Frameworks_Assignment
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Place `metadata.csv` in the `data/` folder.

---

## 📊 Usage

Run the Jupyter Notebook:

```bash
jupyter notebook notebooks/analysis.ipynb
```

Run the Streamlit app:

```bash
streamlit run app/app.py
```
